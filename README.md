### Book Drop
Library management system

#### Features
- Book management (adding, editing)
- Client management
- Book lending and returns
- Rental history

#### Quick start
1. git clone git@github.com:mikhailvzhzhv/book-drop.git
2. ```cd book-drop/```
3. ```cd deployment/```
4. ```docker compose up -d```
5. go to browser http://localhost:8080/
